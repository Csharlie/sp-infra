# apps/

Runnable infra presets — local environment configuration templates.

## Purpose

Pre-configured setups for specific runtime environments. Not application code — infrastructure recipes.

## Planned presets

- `wamp/` — Apache vhost template, WAMP setup guide

## Naming rule

- **apps** = runnable presets (config that makes something work)
- **templates** = file-level templates only (copyable starting points)
